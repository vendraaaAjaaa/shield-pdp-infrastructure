# Shield-PDP Stage 7 - Multi-Environment Topology

The `environment-manager` service models subsidiaries, multi-region enterprise footprints, hybrid cloud, remote workforce, VPN trust, and vendor/supply-chain links.

Topologies:
- `multi-region-enterprise`: Jakarta, shared services, and DR region with subsidiary and vendor trust.
- `hybrid-cloud-workforce`: on-prem, cloud-shared, and remote workforce zones.

Outputs include tenant isolation data, cross-environment telemetry markers, and multi-environment attack graph summaries for SOC and executive reporting.
