# Terminology

| Term | Meaning |
| --- | --- |
| Cyber range | Controlled training environment for realistic security exercises. |
| Purple team | Collaborative red-team and blue-team validation workflow. |
| Synthetic activity | Generated lab telemetry that does not represent real user behavior. |
| Replay | Deterministic reconstruction of events, alerts, and state. |
| Beacon simulation | Safe heartbeat and task-polling model without malware behavior. |
| Persistence simulation | Reversible marker workflow that does not modify host persistence. |
| Attack graph | Graph of identities, services, policies, telemetry, and trust paths. |
| Telemetry SLA | Simulated measure of observability quality and correlation readiness. |
| Policy drift | Difference between expected governance policy and current simulated state. |
| Zero trust simulation | Policy and mTLS posture modeling without real mesh enforcement. |
| Lab secret | Synthetic or generated secret used only inside the isolated lab. |
